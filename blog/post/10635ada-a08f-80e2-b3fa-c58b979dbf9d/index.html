Cursor 대안 → Aider + Supermaven + Continue.dev

2024-06-01 | react


Aider, Supermaven, Continue.dev를 조합하여 효율적으로 사용하는 방법을 설명하겠습니다. 각각의 도구는 다양한 기능을 제공하므로, 이들의 장점을 최대한 활용하여 개발 워크플로우를 최적화하는 것입니다.

도구별 용도 및 강점:

  • Aider:
    1. 코드 리뷰 및 품질 검사: 코드의 스타일, 성능, 보안 등을 자동으로 검토하고 개선 사항을 제시합니다.
    2. 자동 테스트 생성 및 디버깅: 테스트 케이스를 자동으로 생성하고, 오류를 찾아 수정합니다.
  • Supermaven:
    1. AI 비서 기능: 코드 작성 중 발생하는 다양한 질문에 즉각적인 답변을 제공합니다.
    2. 문맥 인식: 특정 코드 섹션의 문제 해결 및 개선 사항을 제시합니다.
  • Continue.dev:
    1. 자동 코드 완성 및 리팩토링: 자동 코드 완성 제안과 리팩토링 제안으로 생산성을 높입니다.
    2. 실시간 협업 기능: 팀원들과 실시간으로 협업하고 코드 리뷰를 진행할 수 있습니다.

    • 효율적 사용 전략:

    각 도구의 강점을 바탕으로, 개발 프로세스에서 어떤 단계에 어떻게 사용할지에 따라 효율성을 높일 수 있습니다.

    1. 코드 작성 단계:

  • Continue.dev:
    • 자동 코드 완성: 코드 작성 중 Continue.dev의 자동 완성 기능을 활용하여 빠르고 정확하게 코드를 작성합니다.
    • 리팩토링 제안: Continue.dev의 실시간 리팩토링 제안을 참고하여 코드 품질을 향상시킵니다.
  • Supermaven:
    • AI 비서 기능: 코드 작성 중 문제가 발생하거나, 특정 기능 구현에 대한 도움이 필요할 때 Supermaven을 통해 즉각적인 질문과 답변을 받습니다.
    • 문맥 인식: 잘못된 부분이나 더 좋은 방법에 대해 Supermaven으로부터 구체적인 피드백을 받습니다.

    • 2. 코드 검토 및 품질 유지 단계:

  • Aider:
    • 코드 리뷰 및 품질 검사: 코드를 커밋하거나 PR을 생성할 때 Aider를 통해 자동으로 코드 리뷰를 실시합니다.
    • 보안 및 성능 검사: 보안 취약점 및 성능 문제를 탐지하고 개선 사항을 제시합니다.
    • 테스트 생성 및 디버깅: 새로운 기능이나 수정한 코드에 대해 자동으로 테스트 케이스를 생성하고, 디버깅을 수행합니다.

    • 3. 협업 및 피드백 단계:

  • Continue.dev:
    • 실시간 협업: 팀원들과 실시간으로 협업하여 코드 리뷰를 진행하고 피드백을 주고받으며 논의합니다.
    • 코드 리뷰: Continue.dev의 기능을 활용해 팀원들과 효율적으로 코드 리뷰를 수행합니다.
  • Supermaven:
    • 추가적인 질문 및 지원: 협업 과정에서 발생하는 추가적인 질문이나 의문 사항에 대해 Supermaven을 통해 빠르게 해결합니다.

    • 워크플로우 예시:

  • 코드 작성 (Continue.dev + Supermaven):
    1. Continue.dev의 자동 코드 완성 및 리팩토링 제안을 활용하여 빠르게 코드를 작성합니다.
    2. Supermaven을 통해 코드 작성 중 발생하는 질문에 대해 즉각적인 도움을 받습니다.
  • 코드 검토 (Aider):
    1. 코드 작성 후 커밋하거나 PR을 만들면 Aider가 자동으로 코드 리뷰를 수행합니다.
    2. Aider가 제시한 리뷰 결과와 개선 사항을 반영하여 코드를 수정합니다.
  • 협업 및 피드백 (Continue.dev + Supermaven):
    1. Continue.dev를 통해 팀원들과 실시간 협업을 진행합니다.
    2. 필요할 경우 Supermaven을 통해 추가 질문이나 문제 해결을 진행합니다.

    • 이러한 식으로 각 도구를 개발 프로세스의 각 단계에서 적절히 활용하면, 개발 속도와 코드 품질을 동시에 향상시킬 수 있습니다. 각각의 도구가 제공하는 기능을 최대한 활용하여 통합된 개발 환경을 구축하세요.
